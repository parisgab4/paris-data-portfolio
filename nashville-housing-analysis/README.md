# Nashville Housing Market Analysis

**Tools:** Python, pandas, matplotlib, Tableau  
**Dataset Size:** ~15,000 records (sampled / synthetic for practice)

## Overview
This project analyzes housing sales in Nashville to understand price trends, neighborhood differences, and factors that impact home value.

## Key Tasks
- Cleaned and transformed raw housing data using pandas
- Engineered features (price per sq ft, house age, renovation age)
- Built visualizations to compare neighborhoods and trends
- Exported cleaned data to Tableau for interactive dashboards

## Insights
- Certain neighborhoods show higher average sale prices and stronger year-over-year growth
- Price per square foot is strongly related to location and house age
- Recently renovated homes tend to sell at a higher price per square foot

## Files
- `Nashville_Housing_Analysis.ipynb` — Full analysis
